Wrapping Up the Path Integral

Last time, we finally defined the path integral. So far we’ve only used it to calculate the probability for a particle to move between two positions x1x_1x1​ and x2x_2x2​. Today, we will generalize this to calculate the probability for a particle to move from any state S1S_1S1​ to any other state S2S_2S2​. We’ll also take the classical limit and derive the cornerstone of classical physics: the principle of least action.

From Position, Everything

Last time, we used defined the path integral as

∫dy(t)=lim⁡N→∞(imℏN2πt)N/2Πa=1N∫−∞∞dya\int dy(t) = \lim_{N \to \infty} \left(\frac{i m \hbar N}{2 \pi t}\right)^{N / 2} \Pi_{a=1}^{N} \int_{-\infty}^{\infty} d y_a∫dy(t)=N→∞lim​(2πtimℏN​)N/2Πa=1N​∫−∞∞​dya​

This allowed us to calculate the probability of a particle moving from position x1x_1x1​ to x2x_2x2​ over a time 0→T0 \to T0→T:

⟨x2∣eiE^ℏt∣x1⟩=∫eiℏ∫0TL(y(λ),y˙(λ))dλdy(t)\bra{x_2} e^{i \frac{\hat{E}}{\hbar} t} \ket{x_1} = \int e^{\frac{i}{\hbar} \int_0^T L(y(\lambda), \dot{y}(\lambda)) d \lambda} dy(t)⟨x2​∣eiℏE^​t∣x1​⟩=∫eℏi​∫0T​L(y(λ),y˙​(λ))dλdy(t)

By itself, this is useful enough. The problem of “where is a particle going” has whole fields of physics dedicated to it, and the path integral is a powerful tool to answer that question. However, we can use it to bootstrap answers to an even more powerful question: what is the probability that a particle will go from any state to any other state?

Let’s write this question using our formalism. Say we have an system that starts off in the state ∣S1⟩\ket{S_1}∣S1​⟩ at time 000. We want to find the probability that it’ll evolve to the state ∣S2⟩\ket{S_2}∣S2​⟩ at time TTT. This would be written as

⟨S2∣eiE^ℏt∣S1⟩\bra{S_2} e^{i \frac{\hat{E}}{\hbar} t} \ket{S_1}⟨S2​∣eiℏE^​t∣S1​⟩

This expression suffers from the same problem as its position version: the long time gap makes the operator in the middle very complicated. One might then ask: could we pull the same trick here, and split this path up into several tiny paths between intermediate states?

The answer is: yes, but it isn’t very helpful. There are two reasons for this:

  1. The operator we’re trying to simplify, eiE^ℏte^{i \frac{\hat{E}}{\hbar} t}eiℏE^​t, is made up of position and momentum operators. Thus we won’t be able to simplify it on the intermediate states ∣Sa⟩\ket{S_a}∣Sa​⟩.
  2. We won’t be able to pull the same trick of inserting momentum basis states, since they probably won’t have a simple dot product with the ∣Sa⟩\ket{S_a}∣Sa​⟩s.

Thus we can’t simply apply the path integral trick to evolution between any states. However, we can do something almost as good…

Bootstrapping From Position Integrals

We can get the more general problem into “path integral form” by using a trick we’ve pulled a couple times now. We can insert 111, in the form of the complete set of position states:

∫−∞∞∣x⟩⟨x∣dx=1\int_{-\infty}^{\infty} \ket{x} \bra{x} dx = 1∫−∞∞​∣x⟩⟨x∣dx=1

By doing this twice, once for each operator, we end up with

⟨S2∣eiE^ℏt∣S1⟩=∫−∞∞∫−∞∞⟨S2∣∣x2⟩⟨x2∣eiE^ℏt∣x1⟩⟨x1∣∣S1⟩dx1dx2\bra{S_2} e^{i \frac{\hat{E}}{\hbar} t} \ket{S_1} = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} \bra{S_2} \ket{x_2} \bra{x_2} e^{i \frac{\hat{E}}{\hbar} t} \ket{x_1} \bra{x_1} \ket{S_1} dx_1 dx_2⟨S2​∣eiℏE^​t∣S1​⟩=∫−∞∞​∫−∞∞​⟨S2​∣∣x2​⟩⟨x2​∣eiℏE^​t∣x1​⟩⟨x1​∣∣S1​⟩dx1​dx2​

Suddenly, this is looking a lot more path integral-y. We can make it look even closer by realizing that ⟨x1∣∣S1⟩\bra{x_1} \ket{S_1}⟨x1​∣∣S1​⟩ is just the x1x_1x1​ component of ∣S2⟩\ket{S_2}∣S2​⟩ (that is, the scalar S2(x)S_2(x)S2​(x)), and similarly for ⟨S2∣∣x2⟩\bra{S_2} \ket{x_2}⟨S2​∣∣x2​⟩:

⟨S2∣eiE^ℏt∣S1⟩=∫−∞∞∫−∞∞S2∗(x2)S1(x1)⟨x2∣eiE^ℏt∣x1⟩dx1dx2\bra{S_2} e^{i \frac{\hat{E}}{\hbar} t} \ket{S_1} = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} S_2^*(x_2) S_1(x_1) \bra{x_2} e^{i \frac{\hat{E}}{\hbar} t} \ket{x_1} dx_1 dx_2⟨S2​∣eiℏE^​t∣S1​⟩=∫−∞∞​∫−∞∞​S2∗​(x2​)S1​(x1​)⟨x2​∣eiℏE^​t∣x1​⟩dx1​dx2​

This is the scaled path integral. It’s close enough to the path integral method that we can usually calculate it with similar methods. In the next section, we will see how we can simplify this even further. For now it suffices to know that inserting position states allows us to calculate any probability density with a path integral-ish approach.

Quantum to Classical

We will wrap up our discussion of the path integral with the small task of showing how it gives rise to all of classical mechanics. Incredibly, this is only a short step from where we already are. We already know that a particle’s probability density to go from x1x_1x1​ to x2x_2x2​ in time TTT is

⟨x2∣eiE^ℏT∣x1⟩=∫eiℏ∫0TL(y(λ),y˙(λ))dλdy(t)\bra{x_2} e^{i \frac{\hat{E}}{\hbar} T} \ket{x_1} = \int e^{\frac{i}{\hbar} \int_{0}^{T} L(y(\lambda), \dot{y}(\lambda)) d \lambda}dy(t)⟨x2​∣eiℏE^​T∣x1​⟩=∫eℏi​∫0T​L(y(λ),y˙​(λ))dλdy(t)

The classical limit of quantum mechanics is when we take ℏ→0\hbar \to 0ℏ→0. However, if we do this immediately, the integral will blow up. Instead we have to use a trick we discussed last time: we need to do a Wick rotation in which we replace λ\lambdaλ (the time variable) with iλi \lambdaiλ. Doing so gives us the integral

∫e−1ℏ∫0TL(y(λ),y˙(λ))dλdy(t)\int e^{-\frac{1}{\hbar} \int_{0}^{T} L(y(\lambda), \dot{y}(\lambda)) d \lambda} dy(t)∫e−ℏ1​∫0T​L(y(λ),y˙​(λ))dλdy(t)

Now the limit is well-defined. Since the exponent gets arbitrarily close to −∞-\infty−∞, the probability density will be dominated by whichever path y(λ)y(\lambda)y(λ) makes the value ∫0TL(y(λ),y˙(λ))dλ\int_{0}^{T} L(y(\lambda), \dot{y}(\lambda)) d \lambda∫0T​L(y(λ),y˙​(λ))dλ as small as possible.

Let me say that again: in the classical limit, the probability density is dominated by the path y(t)y(t)y(t) that minimizes the following integral:

∫0TL(y(λ),y˙(λ))dλ\int_{0}^{T} L(y(\lambda), \dot{y}(\lambda)) d \lambda∫0T​L(y(λ),y˙​(λ))dλ

This is exactly the principle of least action from classical mechanics. From this simple principle – that a particle takes the path of least action – literally all of classical physics can be derived.
