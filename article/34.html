Forces From the Path Exponent (3)

Last time, we attempted to more rigorously derive an integral related to forces in the path integral. Today, we will evaluate that integral.

Into Spherical Coordinates

Through a series of mathematically dubious shenanigans, we have reduced the path integral exponent integral

1(2π)4∫J(p)†1p2−m2+iϵJ(p)d4p\frac{1}{(2 \pi)^4} \int J(p)^\dag \frac{1}{p^2 - m^2 + i \epsilon} J(p) d^4 p(2π)41​∫J(p)†p2−m2+iϵ1​J(p)d4p

for a vacuum perturbation J(p)J(p)J(p) that consists of two spike functions at spacetime points x1,x2x_1, x_2x1​,x2​:

J(x)=δ(x−x1)+δ(x−x2)J(x) = \delta(x - x_1) + \delta(x - x_2)J(x)=δ(x−x1​)+δ(x−x2​)

to an integral over the three spatial components of momentum:

−1(2π)2∫e−i(x2−x1)λpλpλ2+m2+iϵd3pλ-\frac{1}{(2 \pi)^2} \int \frac{e^{-i (x_2 - x_1)_\lambda p_\lambda}}{p_\lambda^2 + m^2 + i \epsilon} d^3 p_\lambda−(2π)21​∫pλ2​+m2+iϵe−i(x2​−x1​)λ​pλ​​d3pλ​

Since the denominator is now always positive, we can finally drop the singularity-dodging iϵi \epsiloniϵ term:

−1(2π)2∫e−i(x2−x1)λpλpλ2+m2d3pλ-\frac{1}{(2 \pi)^2} \int \frac{e^{-i (x_2 - x_1)_\lambda p_\lambda}}{p_\lambda^2 + m^2} d^3 p_\lambda−(2π)21​∫pλ2​+m2e−i(x2​−x1​)λ​pλ​​d3pλ​

We will now evaluate this integral.

First, we’ll swap to a better set of coordinates. Note that the three momentum components px,py,pzp_x, p_y, p_zpx​,py​,pz​ never show up alone. We only have terms containing the magnitude of pλp_\lambdapλ​ and the dot product of pλp_\lambdapλ​ with another vector. This is a strong hint that the integral may be better computed in spherical coordinates.

We’ll define these spherical coordinates as such. The radial coordinate, PPP, will just be the magnitude of pλp_\lambdapλ​. This gives us an easy way to write the denominator.

Next, we need to find a way to write the dot product. We’ll do this by defining a new vector along the displacement:

xˉ:=(x2−x1)λ\bar{x} := (x_2 - x_1)_\lambdaxˉ:=(x2​−x1​)λ​

We’ll set our coordinates such that xˉ\bar{x}xˉ always points in the radial direction. We’ll then define the first angular coordinate, θ\thetaθ, as the angle between the three-momentum pλp_\lambdapλ​ and xˉ\bar{x}xˉ. This will allow us to write their dot product as

∣xˉ∣Pcos⁡(θ)=(x2−x1)λpλ|\bar{x}| P \cos(\theta) = (x_2 - x_1)_\lambda p_\lambda∣xˉ∣Pcos(θ)=(x2​−x1​)λ​pλ​

Finally, we’ll throw in the last unused spherical coordinate ϕ\phiϕ.

In the end, remembering to throw in the area factor, our transformed integral will look like this:

−1(2π)2∫e−i(x2−x1)λpλpλ2+m2d3pλ=−1(2π)2∫0∞∫0π∫02πe−i∣xˉ∣Pcos⁡(θ)P2+m2P2cos⁡(θ)dPdθdϕ-\frac{1}{(2 \pi)^2} \int \frac{e^{-i (x_2 - x_1)_\lambda p_\lambda}}{p_\lambda^2 + m^2} d^3 p_\lambda = -\frac{1}{(2 \pi)^2} \int_0^{\infty} \int_{0}^{\pi} \int_{0}^{2 \pi} \frac{e^{-i |\bar{x}| P \cos(\theta)}}{P^2 + m^2} P^2 \cos(\theta) dP d\theta d\phi−(2π)21​∫pλ2​+m2e−i(x2​−x1​)λ​pλ​​d3pλ​=−(2π)21​∫0∞​∫0π​∫02π​P2+m2e−i∣xˉ∣Pcos(θ)​P2cos(θ)dPdθdϕ

Evaluating The Integral

Phi

We’ll start where we usually start with spherical integrals: doing the trivial ϕ\phiϕ integral:

−1(2π)2∫0∞∫0π∫02πe−i∣xˉ∣Pcos⁡(θ)P2+m2P2cos⁡(θ)dPdθdϕ=−12π∫0∞∫0πe−i∣xˉ∣Pcos⁡(θ)P2+m2P2sin⁡(θ)dPdθ-\frac{1}{(2 \pi)^2} \int_0^{\infty} \int_{0}^{\pi} \int_{0}^{2 \pi} \frac{e^{-i |\bar{x}| P \cos(\theta)}}{P^2 + m^2} P^2 \cos(\theta) dP d\theta d\phi = -\frac{1}{2 \pi} \int_0^{\infty} \int_{0}^{\pi} \frac{e^{-i |\bar{x}| P \cos(\theta)}}{P^2 + m^2} P^2 \sin(\theta) dP d\theta−(2π)21​∫0∞​∫0π​∫02π​P2+m2e−i∣xˉ∣Pcos(θ)​P2cos(θ)dPdθdϕ=−2π1​∫0∞​∫0π​P2+m2e−i∣xˉ∣Pcos(θ)​P2sin(θ)dPdθ

Theta

We’ll then make another change of coordinates. Since θ\thetaθ only shows up in our original integrand as cos⁡(θ)\cos(\theta)cos(θ), we can use this itself as a coordinate

C:=cos⁡(θ)C := \cos(\theta)C:=cos(θ)

We can calculate its differential:

dCdθ=−sin⁡(θ)  ⟹  dC=−sin⁡(θ)dθ  ⟹  dθ=−dCsin⁡(θ)\frac{d C}{d \theta} = -\sin(\theta) \implies dC = -\sin(\theta) d\theta \implies d\theta = -\frac{d C}{\sin(\theta)}dθdC​=−sin(θ)⟹dC=−sin(θ)dθ⟹dθ=−sin(θ)dC​

Behold! The factor of 1sin⁡(theta)\frac{1}{\sin(theta)}sin(theta)1​ in the differential perfectly cancels out the sin⁡(θ)\sin(\theta)sin(θ) from the earlier area factor. This is a common trick in spherical integrals: if θ\thetaθ only shows up via cos⁡(θ)\cos(\theta)cos(θ), this change of coordinates is often useful.

Plugging this in, our integral gets a lot simpler:

−12π∫0∞∫0πe−i∣xˉ∣Pcos⁡(θ)P2+m2P2sin⁡(θ)dPdθ=12π∫0∞∫1−1e−i∣xˉ∣PCP2+m2P2dPdC-\frac{1}{2 \pi} \int_0^{\infty} \int_{0}^{\pi} \frac{e^{-i |\bar{x}| P \cos(\theta)}}{P^2 + m^2} P^2 \sin(\theta) dP d\theta = \frac{1}{2 \pi} \int_0^{\infty} \int_{1}^{-1} \frac{e^{-i |\bar{x}| P C}}{P^2 + m^2} P^2 dP dC−2π1​∫0∞​∫0π​P2+m2e−i∣xˉ∣Pcos(θ)​P2sin(θ)dPdθ=2π1​∫0∞​∫1−1​P2+m2e−i∣xˉ∣PC​P2dPdC

Flipping the bounds of CCC re-introduces our minus sign:

12π∫0∞∫1−1e−i∣xˉ∣PCP2+m2P2dPdC=−12π∫0∞∫−11e−i∣xˉ∣PCP2+m2P2dPdC\frac{1}{2 \pi} \int_0^{\infty} \int_{1}^{-1} \frac{e^{-i |\bar{x}| P C}}{P^2 + m^2} P^2 dP dC = -\frac{1}{2 \pi} \int_0^{\infty} \int_{-1}^{1} \frac{e^{-i |\bar{x}| P C}}{P^2 + m^2} P^2 dP dC2π1​∫0∞​∫1−1​P2+m2e−i∣xˉ∣PC​P2dPdC=−2π1​∫0∞​∫−11​P2+m2e−i∣xˉ∣PC​P2dPdC

The CCC integral is now very simple:

∫−11e−i∣xˉ∣PCP2+m2P2dC=[P2−i∣xˉ∣Pe−i∣xˉ∣PCP2+m2]−11=[P−i∣xˉ∣e−i∣xˉ∣PCP2+m2]−11=P−i∣xˉ∣e−i∣xˉ∣P−ei∣xˉ∣PP2+m2\int_{-1}^{1} \frac{e^{-i |\bar{x}| P C}}{P^2 + m^2} P^2 dC = \left[\frac{P^2}{-i |\bar{x}| P} \frac{e^{-i |\bar{x}| P C}}{P^2 + m^2}\right]_{-1}^{1} = \left[\frac{P}{-i |\bar{x}|} \frac{e^{-i |\bar{x}| P C}}{P^2 + m^2}\right]_{-1}^{1} = \frac{P}{-i |\bar{x}|} \frac{e^{-i |\bar{x}| P} - e^{i |\bar{x}| P}}{P^2 + m^2}∫−11​P2+m2e−i∣xˉ∣PC​P2dC=[−i∣xˉ∣PP2​P2+m2e−i∣xˉ∣PC​]−11​=[−i∣xˉ∣P​P2+m2e−i∣xˉ∣PC​]−11​=−i∣xˉ∣P​P2+m2e−i∣xˉ∣P−ei∣xˉ∣P​

P

All of a sudden, this numerator is something very familiar. The difference of two imaginary exponents like this is just twice the imaginary sine of the argument:

eix−e−ix=(cos⁡(x)+isin⁡(x))−(cos⁡(x)−isin⁡(x))=2isin⁡(x)e^{i x} - e^{-i x} = (\cos(x) + i \sin(x)) - (\cos(x) - i \sin(x)) = 2 i \sin(x)eix−e−ix=(cos(x)+isin(x))−(cos(x)−isin(x))=2isin(x)

Substituting this in, we get a (relatively) simpler expression:

−12π∫0∞P−i∣xˉ∣e−i∣xˉ∣P−ei∣xˉ∣PP2+m2dP=−12π∫0∞P−i∣xˉ∣−2isin⁡(xˉP)P2+m2dP-\frac{1}{2 \pi} \int_0^{\infty} \frac{P}{-i |\bar{x}|} \frac{e^{-i |\bar{x}| P} - e^{i |\bar{x}| P}}{P^2 + m^2} dP = -\frac{1}{2 \pi} \int_0^{\infty} \frac{P}{-i |\bar{x}|} \frac{-2 i \sin(\bar{x} P)}{P^2 + m^2} dP−2π1​∫0∞​−i∣xˉ∣P​P2+m2e−i∣xˉ∣P−ei∣xˉ∣P​dP=−2π1​∫0∞​−i∣xˉ∣P​P2+m2−2isin(xˉP)​dP

Taking a second to group the constants:

−12π∫0∞P−i∣xˉ∣−2isin⁡(xˉP)P2+m2dP=−1π∣xˉ∣∫0∞Psin⁡(xˉP)P2+m2dP-\frac{1}{2 \pi} \int_0^{\infty} \frac{P}{-i |\bar{x}|} \frac{-2 i \sin(\bar{x} P)}{P^2 + m^2} dP = -\frac{1}{\pi |\bar{x}|} \int_0^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} dP−2π1​∫0∞​−i∣xˉ∣P​P2+m2−2isin(xˉP)​dP=−π∣xˉ∣1​∫0∞​PP2+m2sin(xˉP)​dP

We can now pull a second clever trick to turn this sine back into an exponential. The complex exponential eixe^{i x}eix is cos⁡(x)+isin⁡(x)\cos(x) + i \sin(x)cos(x)+isin(x). We already have sin⁡(x)\sin(x)sin(x) here, so if we could add a term proportional to cos⁡(x)\cos(x)cos(x), we could pull everything back into an easier-to-integrate exponential. However, obviously, we can’t add random terms to a random integral… or can we?

Let’s be explicit about what we want. We’d like to be able to write something like

−1π∣xˉ∣∫0∞Psin⁡(xˉP)P2+m2dP=−1π∣xˉ∣∫0∞Psin⁡(xˉP)−icos⁡(xˉ)P2+m2dP=−1π∣xˉ∣(∫0∞Psin⁡(xˉP)P2+m2−i∫0∞Pcos⁡(xˉP)P2+m2dP)-\frac{1}{\pi |\bar{x}|} \int_0^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} dP = -\frac{1}{\pi |\bar{x}|} \int_0^{\infty} P \frac{\sin(\bar{x} P) - i \cos(\bar{x})}{P^2 + m^2} dP = -\frac{1}{\pi |\bar{x}|} (\int_0^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} - i \int_0^{\infty} P \frac{\cos(\bar{x} P)}{P^2 + m^2} dP)−π∣xˉ∣1​∫0∞​PP2+m2sin(xˉP)​dP=−π∣xˉ∣1​∫0∞​PP2+m2sin(xˉP)−icos(xˉ)​dP=−π∣xˉ∣1​(∫0∞​PP2+m2sin(xˉP)​−i∫0∞​PP2+m2cos(xˉP)​dP)

By starting with our dream scenario, we’ve narrowed down our goal. We now just need to show that this second integral is zero.

Unfortunately for us, this integral is not zero. Fortunately for us, it can be convinced to become so with a little trickery.

Consider the original integral again:

−1π∣xˉ∣∫0∞Psin⁡(xˉP)P2+m2dP-\frac{1}{\pi |\bar{x}|} \int_0^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} dP−π∣xˉ∣1​∫0∞​PP2+m2sin(xˉP)​dP

Note that the integrand here is the product of an odd function PPP, another odd function sin⁡\sinsin, and an even function 1P2+m2\frac{1}{P^2 + m^2}P2+m21​. Thus this entire integral is even. If we replaced the sine with a cosine, it would become odd. And if we integrate an odd function over symmetric bounds, it has to be zero!

Let’s put this plan into action. First, we use the even nature of this integral to extend it from 0→∞0 \to \infty0→∞ to −∞→∞-\infty \to \infty−∞→∞:

−1π∣xˉ∣∫0∞Psin⁡(xˉP)P2+m2dP=−12π∣xˉ∣∫−∞∞Psin⁡(xˉP)P2+m2dP-\frac{1}{\pi |\bar{x}|} \int_0^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} dP = -\frac{1}{2 \pi |\bar{x}|} \int_{-\infty}^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} dP−π∣xˉ∣1​∫0∞​PP2+m2sin(xˉP)​dP=−2π∣xˉ∣1​∫−∞∞​PP2+m2sin(xˉP)​dP

This time, when we add our corresponding cosine function, the integrand becomes odd and gets integrated over symmetric bounds – thus becoming zero!

∫−∞∞Pcos⁡(xˉP)P2+m2dP=0\int_{-\infty}^{\infty} P \frac{\cos(\bar{x} P)}{P^2 + m^2} dP = 0∫−∞∞​PP2+m2cos(xˉP)​dP=0

Thus we can rewrite our above integral using a complex exponential:

−12π∣xˉ∣∫−∞∞Psin⁡(xˉP)P2+m2dP=−12π∣xˉ∣∫−∞∞Psin⁡(xˉP)−icos⁡(xˉP)P2+m2dP=i2π∣xˉ∣∫−∞∞PeixˉPP2+m2dP-\frac{1}{2 \pi |\bar{x}|} \int_{-\infty}^{\infty} P \frac{\sin(\bar{x} P)}{P^2 + m^2} dP = -\frac{1}{2 \pi |\bar{x}|} \int_{-\infty}^{\infty} P \frac{\sin(\bar{x} P) - i \cos(\bar{x} P)}{P^2 + m^2} dP = \frac{i}{2 \pi |\bar{x}|} \int_{-\infty}^{\infty} P \frac{e^{i \bar{x} P}}{P^2 + m^2} dP−2π∣xˉ∣1​∫−∞∞​PP2+m2sin(xˉP)​dP=−2π∣xˉ∣1​∫−∞∞​PP2+m2sin(xˉP)−icos(xˉP)​dP=2π∣xˉ∣i​∫−∞∞​PP2+m2eixˉP​dP

Back to Residues

This is now a complex contour integral along a closed(?) contour. Thus it is zero save for the residues.

There are two singularities at play here: P=±imP = \pm i mP=±im. However, for reasons I don’t totally understand, we only consider the singularity at P=+imP = +i mP=+im. The argument is given that “since ∣xˉ∣|\bar{x}|∣xˉ∣ is positive, the function is exponentiall damped in the upper-half plane; thus we want to pick up the pole in the upper half plane.” I don’t understand that at all. Maybe we only want to pick up poles where the function is exponentially damped? But if so, why?

Whatever. Let’s power through:

PeixˉPP2+m2=PeixˉP(P+im)(P−im)  ⟹  Resim=(im)eixˉimim+im=e−xˉm2P \frac{e^{i \bar{x} P}}{P^2 + m^2} = P \frac{e^{i \bar{x} P}}{(P + i m)(P - i m)} \implies \text{Res}_{i m} = (i m) \frac{e^{i \bar{x} i m}}{i m + i m} = \frac{e^{-\bar{x} m}}{2}PP2+m2eixˉP​=P(P+im)(P−im)eixˉP​⟹Resim​=(im)im+imeixˉim​=2e−xˉm​

and the integral as a whole comes out to

where we have finally stopped abusing notation in the exponent and written ∣xˉ∣|\bar{x}|∣xˉ∣ instead of xˉ\bar{x}xˉ. Sorry!

Damn

This is almost what we wanted. We wanted to show that this integral gave rise to a central force law. However, we fell short by the exact factor of 2π2 \pi2π in the denominator that we dropped earlier during the spike function debacle:

−e∣xˉ∣m2∣xˉ∣≠−e∣xˉ∣m4π∣xˉ∣-\frac{e^{|\bar{x}| m}}{2 |\bar{x}|} \neq -\frac{e^{|\bar{x}| m}}{4 \pi |\bar{x}|}−2∣xˉ∣e∣xˉ∣m​=−4π∣xˉ∣e∣xˉ∣m​
