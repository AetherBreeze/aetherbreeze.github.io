To Infinity And Beyond (Again)

Last time, we derived the last fundamental concepts of quantum mechanics in the discrete case. Today, we will generalize these concepts to the continuous case.

What Are Continuous Matrices?

We will start by generalizing our quantum equation of motion:

−iℏddt∣S⟩=E^∣S⟩-i \hbar \frac{d}{dt} \ket{S} = \hat{E} \ket{S}−iℏdtd​∣S⟩=E^∣S⟩

This equation was derived using discrete systems ∣S⟩\ket{S}∣S⟩. In these systems, E^\hat{E}E^ is the energy matrix – a matrix whose eigenvectors are the system’s energy states, and whose eigenvalues are the corresponding energies. However, for continuous systems ∣S(x)⟩\ket{S(x)}∣S(x)⟩, it’s a bit more complicated. We can’t simply write a martix with a continuous number of rows and columns. We’ll have to dig a bit deeper.

Thus, we ask ourselves: what is a matrix?

Perhaps more useful, and in my mind equivalent: what does a matrix do?

This more “functional” question helps a lot. A matrix is not simply a square of numbers. Rather, a matrix M^\hat{M}M^ represents a linear transformation on a vector. A matrix is a function such that for any vectors ∣A⟩,∣B⟩\ket{A}, \ket{B}∣A⟩,∣B⟩ and any constants x,yx, yx,y,

M^(x∣A⟩+y∣B⟩)=xM^∣A⟩+yM^∣B⟩\hat{M}(x \ket{A} + y \ket{B}) = x \hat{M} \ket{A} + y \hat{M} \ket{B}M^(x∣A⟩+y∣B⟩)=xM^∣A⟩+yM^∣B⟩

Read closely, this is important. The above equation is not a property of matrices – it is the definition of a matrix. It just so happens that, when the vectors ∣A⟩,∣B⟩\ket{A}, \ket{B}∣A⟩,∣B⟩ are discrete, all such transformations M^\hat{M}M^ can be written as squares of numbers (in some given basis).

This functional view answers our earlier question: what is a continuous matrix that operates on continuous vectors ∣V(x)⟩\ket{V(x)}∣V(x)⟩? It’s some “operator” M(x)^\hat{M(x)}M(x)^​ that acts on vectors in a way that satisfies

M(x)^(c∣A(x)⟩+d∣B(x)⟩)=cM(x)^∣A(x)⟩+dM(x)^∣B(x)⟩\hat{M(x)}(c \ket{A(x)} + d \ket{B(x)}) = c \hat{M(x)} \ket{A(x)} + d \hat{M(x)} \ket{B(x)}M(x)^​(c∣A(x)⟩+d∣B(x)⟩)=cM(x)^​∣A(x)⟩+dM(x)^​∣B(x)⟩

Matrices and Operators

This generalization is so significant that it’s no longer helpful to call these things “matrices”. The term “matrix” represents that, for discrete systems, a linear transformation can be written as a literal matrix of numbers. Once we leave the discrete, the term becomes at best vestigal; at worst, misleading. Thus we must come up with a new name for these things.

At its core, a “matrix” is a thing that operates on a vector to produce another vector. We will call a general object that acts in this way an operator:

Definition: An operator M^\hat{M}M^ is something that acts on a vector ∣S⟩\ket{S}∣S⟩ to produce a new vector M^∣S⟩\hat{M} \ket{S}M^∣S⟩

Note that this definition is more general than a matrix. We call an operator that satisfies the matrix condition (given above) a linear operator:

Definition: A linear operator M^\hat{M}M^ is an operator that satisfies

M^(c∣A⟩+d∣B⟩)=cM^∣A⟩+dM^∣B⟩\hat{M}(c \ket{A} + d \ket{B}) = c \hat{M} \ket{A} + d \hat{M} \ket{B}M^(c∣A⟩+d∣B⟩)=cM^∣A⟩+dM^∣B⟩

for any vectors ∣A⟩,∣B⟩\ket{A}, \ket{B}∣A⟩,∣B⟩ and constants c,dc, dc,d.

Note that linear operators, like matrices, can have eigenvectors and eigenvalues. A vector ∣S⟩\ket{S}∣S⟩ is an eigenvector of some operator M^\hat{M}M^ iff

M^∣S⟩=m∣S⟩\hat{M} \ket{S} = m \ket{S}M^∣S⟩=m∣S⟩

for some constant mmm, which is called ∣S⟩\ket{S}∣S⟩'s eigenvalue.

Examples of Linear Operators

The best way to remember a new thing is to play around with examples of it. As such, let’s play a little with some examples and understand why they work.

We’ll start with perhaps the simplest linear operator. We can define an operator M(x)^\hat{M(x)}M(x)^​ that simply multiplies the vector by a function:

M(x)^∣S⟩=M(x)∣S⟩\hat{M(x)} \ket{S} = M(x) \ket{S}M(x)^​∣S⟩=M(x)∣S⟩

From the basic properties of multiplication, we can easily see that

M(x)^(c∣A⟩+d∣B⟩)=cM(x)∣A⟩+dM(x)∣B⟩\hat{M(x)} (c \ket{A} + d \ket{B}) = c M(x) \ket{A} + d M(x) \ket{B}M(x)^​(c∣A⟩+d∣B⟩)=cM(x)∣A⟩+dM(x)∣B⟩

and thus this is indeed a linear operator.

A more subtle linear operator, and one we will make extensive use of, is differentiation. Define the operator Dx^\hat{D_x}Dx​^​'s action on a continuous vector ∣S(x)⟩\ket{S(x)}∣S(x)⟩ as

Dx^∣S(x)⟩=ddx∣S(x)⟩\hat{D_x} \ket{S(x)} = \frac{d}{dx} \ket{S(x)}Dx​^​∣S(x)⟩=dxd​∣S(x)⟩

From the properties of differentiation, we can again check that

Dx^(c∣A(x)⟩+d∣B(x)⟩)=ddx(c∣A(x)⟩+d∣B(x)⟩)=cddx∣A(x)⟩+dddx∣B(x)⟩\hat{D_x} (c \ket{A(x)} + d \ket{B(x)}) = \frac{d}{dx} (c \ket{A(x)} + d \ket{B(x)}) = c \frac{d}{dx} \ket{A(x)} + d \frac{d}{dx} \ket{B(x)}Dx​^​(c∣A(x)⟩+d∣B(x)⟩)=dxd​(c∣A(x)⟩+d∣B(x)⟩)=cdxd​∣A(x)⟩+ddxd​∣B(x)⟩

and thus the differentiation operator Dx^\hat{D_x}Dx​^​ is a linear operator.

These are simple examples of linear operators. They are also the only two types of operators used by most in quantum mechanics. Indeed, we have already seen the differentiation operator show up:

iℏddt∣S⟩=E^∣S⟩i \hbar \frac{d}{dt} \ket{S} = \hat{E} \ket{S}iℏdtd​∣S⟩=E^∣S⟩

The operator in this case is

E^=iℏddt\hat{E} = i \hbar \frac{d}{dt}E^=iℏdtd​

If you’ve seen enough background whiteboards in video games, this one might even look a little familiar…

With this definition of “continuous matrices” under our belt, we can turn to generalizing the other new concept: average value matrices. An average value matrix P^\hat{P}P^ for an observable property ppp is one whose eigenvectors are the states with constant values of ppp, and whose eigenvalues are those values of ppp. The energy matrix E^\hat{E}E^ is a specific case of this for the observable property of energy. These matrices are crucial for making experimental predictions, so generalizing them is very important.

How To Measure Continuous Systems

Now that we’ve generalized matrices to linear operators, it’s time to look at their main use in quantum mechanics: average value operators.

Average value operators are the most powerful tool for making predictions using quantum mechanics. An average value matrix P^\hat{P}P^ for some observable property ppp is a matrix whose eigenvectors are the states of constant ppp, and whose eigenvalues are the corresponding values of ppp. We can use this operator to calculate the average value we get when we measure the value of ppp for a system ∣S⟩\ket{S}∣S⟩:

average(P)∣S⟩=⟨S∣P^∣S⟩\text{average}(P)\ket{S} = \bra{S} \hat{P} \ket{S}average(P)∣S⟩=⟨S∣P^∣S⟩

There are two things we need to do to make this continuous. First, we generalize P^\hat{P}P^ from a matrix to a linear operator. Second, since there’s an inner product happening here, we need to express it as an integral:

average(P)∣S(x)⟩=∫S(x)∗P^∣S⟩dx\text{average}(P)\ket{S(x)} = \int S(x)^* \hat{P} \ket{S} dxaverage(P)∣S(x)⟩=∫S(x)∗P^∣S⟩dx

Note that, in accordance with our complex dot product, we have turned ⟨S∣\bra{S}⟨S∣ into the complex conjugate S(x)∗S(x)^*S(x)∗.

The Simplest Average Value Operator

As with linear operators, we will reinforce this concept of continuous systems with some simple examples. We will do so by considering the simplest average value operator: the average value operator for position, X^\hat{X}X^.

This is the simplest operator because we often work in the position basis. For some particle, let ∣S(x)⟩\ket{S(x)}∣S(x)⟩ be the probability density of finding the particle at location xxx. Then the basis vector for a particle at position XXX would just be δ(x−X)\delta(x - X)δ(x−X). To check this, let’s confirm that the dot product of this basis vector with ∣S(x)⟩\ket{S(x)}∣S(x)⟩ extracts S(x)S(x)S(x):

⟨δ(x−X)∣∣S(x)⟩=∫δx−XS(x)dx=S(X)\bra{\delta(x - X)}\ket{S(x)} = \int \delta{x - X} S(x) dx = S(X)⟨δ(x−X)∣∣S(x)⟩=∫δx−XS(x)dx=S(X)

The position operator X^\hat{X}X^ should then be a linear operator such that δx−X\delta{x - X}δx−X is an eigenvector with eigenvalue X. That is, we should have

∫X^δx−X∣S(x)⟩dx=XS(X)\int \hat{X} \delta{x - X} \ket{S(x)} dx = X S(X)∫X^δx−X∣S(x)⟩dx=XS(X)

This is almost too simple – the position operator X^\hat{X}X^ (in the position basis) is just multiplication by xxx!

∫xδx−X∣S(x)⟩dx=XS(X)\int x \delta{x - X} \ket{S(x)} dx = X S(X)∫xδx−X∣S(x)⟩dx=XS(X)

Since this operator simply multiplies by a function, it is trivially a linear operator. The same thing can be generalized to any basis – if the basis vectors are states of constant PPP for some observable PPP, then the average value operator ∣P⟩\ket{P}∣P⟩ will just be ppp.
